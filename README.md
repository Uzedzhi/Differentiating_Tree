# Differentiating Tree

Дифференцирующее бинарное дерево с возможностью чтения математических выражений из файла, созданием красивого HTML и веселого LATEX дампов. 

## Как пользоваться

Вам нужно написать в текстовый файл любое математической выражение, состоящее из 17 стандартных функций
(таких как cos, sin, ^, *, arcsin, arccos, и т.д.). Пример:
```
sin(15 * x^3 + 4) + (cos(10 * x + 5)) ^ 3
```
и дополнить его служебной информацией для построения графика(полный рабочий пример можно найти в [демо-файле](expressions/expression.txt))

Далее запустить программу командой
```
./bin/diff <ваше-имя-файла>.txt
```
И она сама создаст HTML и LATEX дампы, похожие на те из примеров ( [Пример HTML дампа](htmldump/graph.html), [Пример LATEX дампа](latexdump/treelatexdumpEncoded.pdf))
**Настойчиво рекомендую посмотреть содержание LATEX дампа и улыбнуться.**

## Как устроено

В дифференциальном дереве реализовано множество разных функций. Таких как:
  - Рекурсивный спуск для обработки выражения из текстового файла, который преобразует Набор текста в бинарное дерево.
  - Набор оптимизаций дерева (удаление ненужных ветвей, например x + 0 = x и слияние с вычислением всех ветвей, например 1 * 5 + sin(pi) = 5)
  - Дифференцирование дерева с учетом переменных и сложных функций
  - Нахождение ряда Тейлора выражения и преобразования его в дерево
  - Построение графика функции вместе с рядом Тейлора и касательное к графику
  - Вычисление дерева (спрашивая значение всех переменных у пользователя)
  - Генерация полностью рандомного математического уравнения с одной переменной и перевод его в дерево
  - Перевод из дерева обратно в текстовый файл
